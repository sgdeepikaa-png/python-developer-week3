# python-developer-week3
Q1. Why should we use modules?

Modules help us reuse code in different programs. They make the program easier to read, organize, and maintain.

Q2. What is the difference between a dictionary and a list?

A list stores items using indexes, while a dictionary stores values using unique keys. Dictionaries are useful for searching data quickly.

Q3. What did you discover while testing?

The assert statements helped me verify that every function returned the correct output. If a function had an error, the failing assertion helped me identify and fix it.
